# New Contributor Questionnaire

Copy and fill out this template, then share it with me to add the contributor to the website.

---

## Required Information

### 1. Basic Details
```
Full Name: 
Initials: (e.g., SA for Siddharth Agarwal)
Role/Title: (e.g., Developer, UI Designer, CA Contributor)
```

### 2. LinkedIn Profile (Optional)
```
LinkedIn URL: 
```

### 3. Bio (2-4 sentences)
```
Bio: 
(Background, expertise, why they contribute to CA Saathi)
```

### 4. Tools Created/Contributed To
```
Tools: 
(Comma-separated list of tools they built or contributed to)
```

---

## Example (Siddharth Agarwal)

```
Full Name: Siddharth Agarwal
Initials: SA
Role/Title: Founder & Lead Developer

LinkedIn URL: https://www.linkedin.com/in/siddharth-agarwal-a4444a227

Bio: CA Final student from Kolkata who is passionate about leveraging AI-powered solutions to simplify everyday tasks. The project reflects his vision of combining technology with practical professional needs to build tools that genuinely help the CA community.

Tools: Privacy Shield, File Saathi, 26AS Analyzer, LeaseWizard
```
