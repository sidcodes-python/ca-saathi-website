Frequently Asked Questions

Find answers to common questions about CA Saathi, our tools, downloads, and more.

About CA Saathi

What is CA Saathi?

CA Saathi is a free toolkit ecosystem designed specifically for Chartered Accountancy students and professionals. We provide offline-first desktop applications for audit, tax, and compliance workflows. All tools are built with privacy in mind — your data never leaves your computer.

Who created CA Saathi?

CA Saathi was created by Siddharth Agarwal, a CA Final student from Kolkata who is passionate about leveraging AI-powered solutions to simplify everyday tasks. The project reflects his vision of combining technology with practical professional needs to build tools that genuinely help the CA community.

Is CA Saathi affiliated with ICAI or any official body?

No. CA Saathi is an independent, community-driven project. We are not affiliated with, endorsed by, or connected to the Institute of Chartered Accountants of India (ICAI) or any other official regulatory body. Our tools are created independently to help CA students and professionals.

Downloads & Security

Why does my browser show a security warning when downloading?

This is completely normal for any .exe file downloaded from the internet. Browsers like Chrome, Edge, and Firefox display warnings for all executable files as a security precaution — this is not specific to our tools.

Our tools are safe to download and use. To proceed:

  • Chrome: Click the "^" arrow next to the download → Select "Keep"
  • Edge: Click "..." → "Keep" → "Keep anyway"
  • Firefox: The download should proceed normally
Is the downloaded file safe?

Yes. All our tools are:

  • Open-source — you can review the source code on GitHub
  • Built transparently using GitHub Actions (automated builds)
  • Distributed directly from GitHub Releases (no third-party hosting)
  • Offline-first — they don't send data anywhere

If you're still concerned, you can scan any downloaded file with your antivirus software or upload it to VirusTotal for analysis.

Why are tools hosted on GitHub instead of directly on the website?

We use GitHub Releases for several important reasons:

  • Transparency: GitHub shows the complete source code and build process
  • Trust: GitHub is a reputable platform used by millions of developers
  • Version Control: Easy access to all previous versions
  • Reliability: Fast, reliable downloads from GitHub's CDN
What if the download link doesn't work?

If a download link isn't working:

  • Try refreshing the page
  • Check your internet connection
  • Visit the tool's GitHub repository directly
  • Report the issue via our Community Hub

Installation

Do I need to install anything?

Most of our tools are portable applications — just download the .exe file and run it. No installation required. Some tools may require extraction if downloaded as a .zip file.

Why does Windows SmartScreen block the app?

Windows SmartScreen shows warnings for applications from "unknown publishers." This happens because:

  • Our tools are not signed with an expensive code-signing certificate (~$200-400/year)
  • SmartScreen learns from download frequency — less downloaded apps trigger warnings
  • This is a standard Windows security feature, not an indication of malware

As a free, community-driven project, we prioritize keeping tools free over purchasing expensive certificates.

How do I allow the app to run past SmartScreen?

When you see the "Windows protected your PC" message:

  1. Click "More info" (this reveals additional options)
  2. Click "Run anyway"

This is a one-time action. Windows will remember your choice for that specific file.

Usage

Do the tools require an internet connection after download?

No. All CA Saathi tools are designed to work completely offline. Once downloaded, you never need an internet connection to use them. This is intentional — we believe your sensitive financial data should never leave your computer.

What operating systems are supported?

Currently, our tools are built for Windows (Windows 10 and later recommended). We're working on macOS and Linux support for future releases. Check individual tool pages for specific requirements.

Where is my data stored when using the tools?

All data remains on your computer only. Depending on the tool:

  • Processed files are saved wherever you choose to save them
  • Temporary files (if any) are stored locally and can be deleted
  • No data is ever uploaded to any server

Updates

How do I get updates for the tools?

To update a tool:

  1. Visit the tool's page on CA Saathi website
  2. Check the version number against what you have
  3. Download the latest version if there's an update
  4. Replace your old file with the new one

We're working on adding update notifications in future versions.

Will my data be lost when I update a tool?

Generally, no. Since our tools are portable and don't store data inside themselves, updating (replacing the .exe file) won't affect your saved work. Your files remain wherever you saved them on your computer.
