Want to Contribute a Tool?

Open source preferred — We prioritize tools with publicly available source code that the community can audit and trust.

Offline-first design — Tools should work without an internet connection after download. Client data must never be transmitted.

CA-relevant use case — Tools should address real workflows faced by CA students or professionals in audit, tax, or compliance.
